Incorrect Web Root

If you're reading this, it probably means that the webmaster has put their root directory in the wrong place. Firefolio seperates out the files that the user interacts with from the files that make it work on purpose. It stops people from peeking at your configuration files and triggering possibly dangerous scripts without permission.

The server should point to the public_html folder, and not the firefolio folder included in the download.

In any case, the site should be available here.
